honker ships as a SQLite loadable extension plus bindings for Python, Node, Rust, Go, Ruby, Bun, Elixir, C++, and .NET / C#. The extension defines the on-disk format; bindings are thin wrappers that share it.

If you're new, read [Getting started](/getting-started/). Otherwise, pick what you want to do:

## What's not here

Framework helpers for FastAPI/Django/Flask/Express/Rails aren't shipped. Wiring honker into a web framework is about 20 lines of glue; the repo's `examples/` has patterns.

Out of scope on purpose: task pipelines, chains, groups, chords, multi-writer replication, workflow orchestration with DAGs.
